Introduction to C

C is a powerful general-purpose programming language created in 1972. It is widely used for operating systems, memory-efficient applications, and general-purpose software.

Key Features

  • Efficient and simple
  • Procedural language
  • Low-level memory access with pointers
  • Highly portable

History of C

C was developed by Dennis Ritchie at Bell Labs in the early 1970s. It has since become one of the most widely used programming languages, with many other languages, including C++, C#, and Java, derived from it.

Basic Structure of a C Program

This is the basic structure of a C program:


#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Why Learn C?

C is the foundation for many other languages, enables system-level programming, and allows for efficient performance in various applications.

Quick Quiz

What is the output of the following C program?


#include <stdio.h>

int main() {
    printf("Hello, C!\n");
    return 0;
}
  • A) Hello, World!
  • B) Hello, C!
  • C) Hello!

Answer: B) Hello, C!
